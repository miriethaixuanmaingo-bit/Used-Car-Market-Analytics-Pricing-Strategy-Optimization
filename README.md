## Project Overview
This repository contains an end-to-end data analytics and business intelligence project built for **NextDrive**, a digital used-car marketplace platform expanding across Germany. 

The goal of this project is to transform a high-volume dataset of messy vehicle listings into an executive decision-making framework. By offloading heavy data aggregations to **PostgreSQL (SQL)** and creating an agile, 3-page interactive workspace layout in **Power BI**, this solution provides dealership networks and procurement teams with a complete operational radar system.

###  Key Repository Architecture & Features:
* **SQL_Scripts/**: Production-ready database queries evaluating volume distributions, calculating localized moving statistical means, and filtering multi-brand inventory variables.
* **PowerBI_Dashboard/**: Interactive `.pbix` canvas architecture built with clean data ingestion parameters (`65001: Unicode UTF-8`) and professional UI/UX design (including conditional layout heatmaps and customized KPI cards).
* **Business Use-Cases Addressed**: Exponential asset depreciation modeling, vehicle turnover liquidity tracking, transmission configuration luxury premium decay tracking, and automated real-time procurement arbitrage identification.
